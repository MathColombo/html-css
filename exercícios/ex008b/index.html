Código-fonte e indentação

Para melhorar a fonte de códigos como por exemplo: 

            
    n = int(input('Digite um número para saber se ele é maior que 6: '))
    if n > 6:
        print('Sim, seu número é maior que seis!!')
    elif n = 6:
        print('Seu número é igual a seis')
    else:
        print('Não, seu número não é maior que seis!!')
    print('Programa finalizado...')
utilizamos a tag Code. Já para aplicar a indentação usamos a tag PRE.

Citações e outros

Para fazer uma citação simples utilizamos a Tag Q. Já para fazer uma

citação completa
usamos a tag BLOCKQUOTE e ainda com link da citação.

Para utilizar abv utilizamos a tag ABBR.

E uma das mais inúteis, a tag BDO, que deixa o texto invertido assim: Texto comum.
